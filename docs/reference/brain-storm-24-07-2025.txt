# IAM-CORE

## Frontend: niceGUI
## Backend: fastAPI
## Database: Postgre + pgvector + llama-index
## Deploy: VPS com GNU/Linux Ubuntu Server 24.x
## Linguagem e framework: python, python-dependency-injector (para plugins), agno (para agentes)
## Processador de PDFs com arquitetura baseada no sparrow (estrutura-sparrow-para-agnos.md)

## Primeiro Agente: processador de PDF (ler, transcrever, OCR, cadastrar com nome+cpf+data, hash que só considera o conteúdo do arquivo para evitar redundância)
## Segundo Agente: redator de quesitos judiciais
## Terceiro Agente: chatbot (messenger, instagram, whatsapp, site); speechify; whisper (ou outro do google)
