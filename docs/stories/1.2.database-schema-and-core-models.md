```typescript
interface User {
  user_id: string;
  email: string;
  role: 'sysadmin' | 'admin' | 'user';
  is_active: boolean;
  totp_enabled: boolean;
  created_at: string; // ISO 8601 timestamp

}

interface UserCreate {
  email: string;
  password: string;
  role: 'sysadmin' | 'admin' | 'user';

  role?: 'sysadmin' | 'admin' | 'user';
  is_active?: boolean;
}
```

### Database Schema Requirements
**[Source: architecture/database-schema.md]**

**Required PostgreSQL Extensions:**
- `uuid-ossp` for UUID generation
- `pgcrypto` for cryptographic functions
- `vector` for future RAG processing capabilities

**Users Table Constraints:**
- Email format validation with regex pattern
- Password hash minimum length (60 characters for bcrypt)
- Role check constraint (sysadmin, admin, user)
- TOTP secret storage (32 character Base32)
- Unique email constraint and proper indexing

**Clients Table Constraints:**
- SSN format validation: `^\d{3}-\d{2}-\d{4}$`
- Name length minimum 2 characters after trim
- Birth date range: 1900-01-01 to (current date - 13 years)
- Status check constraint (active, inactive, archived)
- Foreign key references to users table

**Required Indexes:**
- Full-text search index on client names using `gin(to_tsvector('english', full_name))`
- Performance indexes on ssn, status, created_at for clients
- Audit log indexes on table_name and timestamp

### Backend Architecture Integration
**[Source: architecture/backend-architecture.md]**

**SQLModel Integration Pattern:**
```python
class ClientBase(SQLModel):
    """Base client fields for sharing between models"""
    full_name: str = Field(min_length=2, max_length=255)
    ssn: str = Field(regex=r'^\d{3}-\d{2}-\d{4}$')
    birth_date: date
    status: ClientStatus = ClientStatus.ACTIVE
    notes: Optional[str] = Field(default=None, max_length=1000)

class Client(BaseModel, ClientBase, table=True):
    """Client database model"""
    __tablename__ = "clients"
    
    client_id: UUID = Field(primary_key=True, alias="id")
    created_by: UUID = Field(foreign_key="users.user_id")
    updated_by: UUID = Field(foreign_key="users.user_id")
    
    # Relationships
    creator: User = Relationship(back_populates="created_clients")
```

- **Naming Conventions:** Database tables use `snake_case`, API routes use `kebab-case`
