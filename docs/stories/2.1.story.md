# Story 2.1: Aprimoramento da Interface de Gerenciamento de Clientes

## Story
**As a** user (admin_user or common_user), **I want** to see a list of all documents that have been processed for a specific client, along with their status, **so that** I have the necessary context before using a writing agent.

## Acceptance Criteria
1. In the client details page, a new section displays a list of all documents sent for them.
2. The list displays the file name, upload date and current status ("Processando", "Concluído", "Falha").
3. The interface is updated to reflect when a document's status changes.
4. The user can view a summary of extracted data from a "Concluído" document.
